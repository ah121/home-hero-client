## Key Features

| Feature                     | Description                                                                 |
|-----------------------------|-----------------------------------------------------------------------------|
| **Secure Authentication**     | Firebase Auth (Email/Password + Google Sign-In) with persistent login       |
| **Private Routes**            | Protected pages: My Services, Add Service, Bookings, Profile                |
| **Provider Dashboard**        | Add, Update, Delete, and View only your own services                        |
| **Smart Booking System**      | Book any service via modal • Cannot book your own service                  |
| **Rating & Review System**    | Leave star ratings + reviews • Powers "Top 6 Rated" section                 |
| **Dynamic Homepage**          | Animated hero slider • Top 6 Rated • Latest Services                        |
| **Advanced Filtering**        | Search & filter services by category, price, location                      |
| **Light/Dark Mode**           | Full theme toggle with smooth transitions                                   |
| **100% Responsive**           | Flawless on mobile, tablet, and desktop                                     |
